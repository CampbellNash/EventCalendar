EventCalendar for Umbraco

EventCalendar for Umbraco is a package that lets you create multiple calendar with multiple events per calendar. Also you can create locations where your events take place. These locations are reusable and will be visualised with Google Maps.

Installation

Simple download the Package from the project page on our.umbraco.com and install it. In the content section create the EC_Calendar node where you need it. Under the created node create the EC_EventDetails node. It is necessary that the detail page is placed under the calendar node for the package to work right.

Usage

Open the new section created by the plugin. If you can't see it, then you have to set the permissions on your current user. Then start creating locations, calendars and events.
